# Extension Icons

Place your extension icons in this directory with the following sizes:

- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon32.png` - 32x32 pixels (toolbar icon @2x)
- `icon48.png` - 48x48 pixels (extension management)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Icon States

For connection status indication, you can also create:

- `icon16-gray.png` - Disconnected state
- `icon32-gray.png` - Disconnected state
- `icon48-gray.png` - Disconnected state
- `icon128-gray.png` - Disconnected state

- `icon16-red.png` - Error state
- `icon32-red.png` - Error state
- `icon48-red.png` - Error state
- `icon128-red.png` - Error state

## Design Guidelines

- Use the PwdGuard logo/branding
- Keep icons simple and recognizable at small sizes
- Use transparency for better integration
- Follow Chrome extension icon guidelines
